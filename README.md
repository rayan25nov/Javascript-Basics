## 🚀 **Getting Started**

### **Prerequisites**
- Node.js (v16 or higher)
- npm or yarn
- Git

### **Installation**
```bash
# Clone the repository
git clone https://github.com/rayan25nov/Javascript-Basics.git
cd Javascript-Basics

# Install dependencies
npm install

```
